# Challenge_Telecom_X


## Descripción del proyecto
Este proyecto realiza un análisis exploratorio de datos (EDA) sobre la evasión de clientes (Churn) en una empresa de servicios por suscripción.

El objetivo principal es identificar patrones y variables asociadas con la cancelación del servicio, con el fin de generar insights estratégicos que ayuden a reducir la tasa de abandono.

## Objetivos
- Analizar la distribución general del churn.

- Explorar la relación entre variables categóricas y churn.

- Analizar cómo las variables numéricas se distribuyen entre clientes que cancelan y los que no.

- Identificar factores clave que influyen en la evasión.

- Proponer recomendaciones estratégicas basadas en los hallazgos.

## Limpieza y tratamiento de datos
- Estandarización de nombres de columnas.

- Conversión de variables numéricas almacenadas como string.

- Corrección de inconsistencias (ej. clientes con tenure = 0 y total charges vacío → asignado 0).

- Verificación de valores infinitos o anómalos.

## Análisis exploratorio
- Se identificó el porcentaje total de clientes que cancelaron vs los que permanecen activos.
- Se diseñó un análisis de variables categóricas vs churn.
- Se diseñó un análisis de variables numéricas vs churn.

## Conclusiones
- El tiempo de permanencia es un factor determinante en la cancelación.

- Los contratos de corto plazo incrementan el riesgo de evasión.

- El método de pago está asociado con el comportamiento de churn.

- Servicios complementarios parecen reducir la probabilidad de abandono.
